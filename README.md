# EasyEDA
This is an R package and functions for Exploratory data analysis (EDA) in R software.

# Installation in R
# Install devtools
devtools::install_github("hadley/devtools")
# Install EasyEDA
devtools::install_github("FAkohoue/EasyEDA")
